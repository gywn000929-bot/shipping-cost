# 대아홀딩스 수입물류 분석 포털

DAE-A 그룹 수입 자재의 **선박·항공 물동량, 통관 실적, 항공 중량, 출하경비**를 한 곳에서 분석하는 단일 HTML 대시보드입니다.

## 사용법

`index.html`을 브라우저로 열면 됩니다. 별도 서버·빌드 없이 동작하며, 차트는 CDN의 Chart.js를 사용합니다.

## 구성

- **요약** — 개요
- **물동량 추이(주간)** — 선박 추이, 항공 추이, 공장별 분석
- **통관·비용(월간)** — 통관 실적, 항공 중량, 출하경비
- **검색·상세** — 품목 검색, 메이커·거래처
- **확장 분석** — 법인별 통관, 월별 시즌성, 거래처·실효환율
- **심화 분석(통관원장)** — 파레토·ABC, 항공→선박 전환 시뮬레이션, 환율 추이
- **종합** — 인사이트·제언

## 데이터 범위

- 통관: 2025.07~2026.07 · 출하경비: 2025.09~2026.06 · 주간 물동량: 2026.03~07
- 심화 분석은 건별 통관 상세 원장(2025.01~2026.07)을 기반으로 하며, 품목코드 체계(S###-######)가 기존 지표와 동일해 상호 정합적입니다.
